# README#
Este repositorio contiene información turística de Chiloé y sus respectivos comentarios descargados de Google Maps. Creado en Python 3.10.3 y con librerias indicadas en requirements.txt

Carpetas:

- Base de datos: Carpeta en la que se guardan los lugares y comentarios de Google Maps de los diferentes lugares turisticos de Chiloé.
		 También contiene los códigos para descargar los compentarios con las Api's Outscraper y Apify.
